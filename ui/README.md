# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

## Themes

Public-facing pages are compiled with one of two themes:

- `liquid`: default LiquidGlass UI.
- `classic`: non-glass Vant UI with polished motion.

Commands:

```bash
npm run dev
npm run dev:classic
npm run build
npm run build:classic
npm run build:all
```

`npm run build` builds `liquid` for backward compatibility. Theme-specific builds output to `dist/liquid` and `dist/classic`.
